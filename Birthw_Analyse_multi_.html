- Régression logistique binaire BIRTHWT¶

  1. Charger le jeu de donnes 'birthwt' du package 'MASS' dans R.
  2. tude descriptive des diffrentes variables de 'birthwt'.
  3. Diviser la base de donnes en 90% pour l'apprentissage et 10% pour la validation.
  4. Tester l'association de la variable 'low' avec les autres variables par rgression logistique binaire.
  5. valuer les indicateurs de performance de prdiction du modle choisi.
  6. Transformer les variables 'ftv' et 'ptl' en variables qualitatives.
  7. Diviser la nouvelle base de donnes aprs le codage des variables 'ftv' et 'ptl'.
  8. valuer les indicateurs de performance du modle choisi avec le package 'blorr'.

Présentation des donnèes birthwt :¶

  • low : Variable binaire (0 ou 1) indiquant si le poids du bb la naissance est faible (1) ou non (0).

  • age : ge de la mre en annes.

  • lwt : Poids de la mre au dernier cycle menstruel avant la grossesse (en livres).

  • race : Catgorie raciale de la mre (reprsente par des entiers, par exemple 1 pour une race, 2 pour une autre, etc.).

  • smoke : Variable binaire indiquant si la mre fumait (1) ou non (0) pendant la grossesse.

  • ptl : Nombre de grossesses antrieures avec un accouchement prmatur.

  • ht : Variable binaire indiquant si la mre a des antcdents d'hypertension (1) ou non (0).

  • ui : Variable binaire indiquant la prsence (1) ou l'absence (0) d'irritabilit utrine.

  • ftv : Nombre de visites chez le mdecin au premier trimestre de la grossesse.

  • bwt : Poids du bb la naissance (en grammes).

Chaque variable reprsente un facteur potentiel influenant le poids du bb la naissance, utile pour l'analyse statistique et la modlisation.

4. Régression logistique binaire¶

La rgression logistique binaire repose sur le modle logistique, qui utilise la fonction de lien logit pour transformer les probabilits en une relation linaire avec les prdicteurs. Le modle est formul comme suit :

  • log(p / (1 - p)) = 0 + 1X1 + 2X2 + ... + n*Xn

o :

  • p reprsente la probabilit que la variable dpendante soit gale 1 (classe positive).
  • 0 est l'intercept du modle.
  • 1, 2, ..., n sont les coefficients associs aux variables indpendantes X1, X2, ..., Xn. La fonction logit transforme le rapport des probabilits en une relation linaire.
  • La variable "y" tant qualitative donc utilisation la fonction "glm" avec une famille binomiale et une fonction de lien logit pour modliser la relation entre "y" et les autres variables explicatives.
  • Les modles "glm" sont adapts pour traiter des variables de rponse catgorielles, et la famille binomiale avec une fonction de lien logit est couramment utilise pour les variables binaires (0/1).
  • Cette approche permet de modliser de manire flexible les relations complexes entre les variables, ce qui peut tre crucial pour capturer des tendances non linaires ou des interactions significatives dans vos donnes.

Tracer une courbe ROC¶

Spcificit et sensibilit :

  • la sensibilit est : TP / (TP + FN) = TP / P.
  • la spcificit est : TN / (TN + FP) = TN / N.

avec

  • TP (true positives) : les prdits positifs qui le sont vraiment.
  • FP (false positives) : les prdits positifs qui sont en fait ngatifs.
  • TN (true negatives) : les prdits ngatifs qui le sont vraiment.
  • FN (false negatives) : les prdits ngatifs qui sont en fait positifs.
  • P (positives) : tous les positifs quelque soit l'tat de leur prdiction. P = TP + FN.
  • N (negatives) : tous les ngatifs quelque soit l'tat de leur prdiction. N = TN + FP.

La courbe ROC (Receiver Operating Characteristic) illustre la performance d'un modle de classification en affichant

  • la sensibilit par rapport 1 - spcificit diffrents seuils. Typiquement, elle s'tend de (0,0) (1,1), surpassant la ligne diagonale qui reprsente une prdiction alatoire.
  • Plus la courbe s'lve au-dessus de cette diagonale, meilleure est la prdiction.
  • Une courbe idale toucherait le point (0,1), indiquant une sensibilit parfaite sans faux positifs.
  • L'AUC (Area Under the Curve) quantifie cette performance :
    • 1 indique une prdiction parfaite
    • 0,5 quivaut une prdiction alatoire.
